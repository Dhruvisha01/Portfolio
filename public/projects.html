AI Powered Java Tutor

Designed and developed an interactive Java tutor to help community college students master foundational programming concepts. Topics include printing output, user input, loops, conditionals, methods, and arrays. Tailored for beginners, it reinforces learning through guided practice and real coding tasks, highlighting my ability to build accessible educational tools grounded in learning science.

Developed an interactive 3D physics lab using Three.js to enable students in rural India to conduct virtual physics experiments on Android devices. This simulation-based learning tool provides realistic lab experiences without the need for physical equipment. Designed with learning science principles, the lab enhances conceptual understanding through hands-on, inquiry-based learning, making STEM education more accessible and engaging.
